# 19-mayis-anma-web-sitesi-71
19 Mayıs'ın anısına adanmış etkileyici bir web sitesi. Kullanıcıların 19 Mayıs'ın tarihsel önemi hakkında bilgi edinmelerini, anma etkinliklerini takip etmelerini ve katkıda bulunmalarını sağlayan bir platform. Modern UI/UX prensipleriyle tasarlanmış olup, Vite, React, TypeScript, Tailwind CSS, Lucide React ve Framer Motion kullanılarak geliştir...
